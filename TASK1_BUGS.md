## Bug 1 — Объявления из другого города при поиске по Москве

**Severity:** P1 (High)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть страницу поиска объявлений.
2. Установить регион **Москва**.
3. Перейти в категорию **Животные → Кошки → Персидская**.
4. Просмотреть список объявлений.

### Фактический результат

В выдаче присутствует объявление с локацией **Новосибирск**.

### Ожидаемый результат

В результатах поиска должны отображаться объявления **только из выбранного региона (Москва)**.

## Bug 2 — Фильтр пола работает некорректно

**Severity:** P1 (High)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть страницу поиска объявлений.
2. В фильтрах выбрать **Пол → Мальчик**.
3. Просмотреть результаты поиска.

### Фактический результат

В выдаче присутствует объявление **«Персидский котёнок девочка»**.

### Ожидаемый результат

В выдаче должны отображаться **только объявления с полом “Мальчик”**.

## Bug 3 — Фильтр цены работает некорректно

**Severity:** P1 (High)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть страницу поиска объявлений.
2. Установить диапазон цены **1000 — 30000 ₽**.
3. Просмотреть результаты поиска.

### Фактический результат

В выдаче отображаются объявления с ценами:

* **500 ₽**
* **40000 ₽**

что выходит за пределы установленного диапазона.

### Ожидаемый результат

В результатах должны отображаться **только объявления в диапазоне 1000–30000 ₽**.

## Bug 4 — Иконка сердца выполняет неочевидное действие

**Severity:** P2 (Medium)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть страницу поиска объявлений.
2. Навести курсор на иконку **сердца** на карточке объявления.

### Фактический результат

Подсказка указывает действие:

> «Не показывайте мне похожие объявления»

Иконка визуально ассоциируется с **добавлением в избранное**, но выполняет другое действие.

### Ожидаемый результат

Иконка **сердца** должна использоваться для **добавления объявления в избранное** или иметь другой визуальный символ.

## Bug 5 — Карточки объявлений расположены неровно

**Severity:** P3 (Low)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть страницу поиска объявлений.
2. Посмотреть на расположение карточек во **втором ряду**.

### Фактический результат

Карточки расположены **неровно**, расстояние между ними отличается.

### Ожидаемый результат

Карточки должны быть **выровнены по сетке с одинаковыми отступами**.

---

## Bug 6 — Контент объявления не соответствует категории

**Severity:** P2 (Medium)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть категорию **Животные → Кошки**.
2. Просмотреть список объявлений.

### Фактический результат

В выдаче присутствует объявление **«Персидский котёнок плюшевый»**, где изображена **игрушка**, а не живое животное.

### Ожидаемый результат

В категории **Животные** должны отображаться **только реальные животные**.

---

## Bug 7 — Сообщение об ошибке отображается при загруженной выдаче

**Severity:** P2 (Medium)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть страницу поиска объявлений.
2. Посмотреть правую часть страницы.

### Фактический результат

Отображается сообщение:

> «Ошибка, обновите страницу»

При этом список объявлений отображается.

### Ожидаемый результат

Сообщение об ошибке не должно отображаться, если данные загрузились успешно.

## Bug 8 — Неконсистентное отображение даты публикации объявления

**Severity:** P3 (Low)
**Environment:** Web, Desktop, Chrome

### Шаги воспроизведения

1. Открыть страницу поиска объявлений.
2. Просмотреть карточки объявлений.

### Фактический результат

У части объявлений отображается информация о времени публикации (**«7 дней назад»**), а у некоторых карточек эта информация отсутствует.

### Ожидаемый результат

Информация о времени публикации должна **отображаться одинаково для всех объявлений**.
