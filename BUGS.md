# BUGS — найденные дефекты API `qa-internship.avito.com`

Ниже перечислены дефекты, обнаруженные при автоматизации тест-кейсов из `TESTCASES.md`.

---

## BUG-001: `POST /api/1/item` не возвращает созданное объявление (нет `id` и полей), только строку в `status`

- **Серьёзность**: Major (нарушение контракта ответа, ломает интеграции)
- **Окружение**: `https://qa-internship.avito.com`, дата: 2026-04-18
- **Шаги воспроизведения**:
  - Отправить `POST https://qa-internship.avito.com/api/1/item` с валидным JSON:
    - `sellerID` (int), `name` (string), `price` (int), `statistics` (object)
- **Фактический результат**:
  - `200 OK`
  - Тело ответа: `{"status": "Сохранили объявление - <uuid>"}` (без `id`, `sellerId`, `name`, `price`, `statistics`, `createdAt`)
- **Ожидаемый результат** (по Postman-коллекции):
  - `200 OK`
  - Тело ответа содержит объект созданного объявления с `id`, `sellerId`, `name`, `price`, `statistics`, `createdAt`

---

## BUG-002: `price = 0` трактуется как “поле price обязательно” (валидация неверная)

- **Серьёзность**: Major (некорректная обработка граничного значения)
- **Окружение**: `https://qa-internship.avito.com`, дата: 2026-04-18
- **Шаги воспроизведения**:
  - Отправить `POST https://qa-internship.avito.com/api/1/item` с `price: 0`
- **Фактический результат**:
  - `400 Bad Request`
  - Сообщение: `поле price обязательно`
- **Ожидаемый результат**:
  - `200 OK`
  - Объявление создаётся с `price = 0`

---

## BUG-003: Нулевая статистика (`likes/viewCount/contacts = 0`) отклоняется как “поле likes обязательно”

- **Серьёзность**: Major (некорректная обработка граничного значения)
- **Окружение**: `https://qa-internship.avito.com`, дата: 2026-04-18
- **Шаги воспроизведения**:
  - Отправить `POST https://qa-internship.avito.com/api/1/item` со статистикой:
    - `{"contacts": 0, "likes": 0, "viewCount": 0}`
- **Фактический результат**:
  - `400 Bad Request`
  - Сообщение: `поле likes обязательно`
- **Ожидаемый результат**:
  - `200 OK`
  - Нулевая статистика должна сохраняться

---

## BUG-004: Отрицательная цена (`price < 0`) принимается (должна быть отклонена)

- **Серьёзность**: Medium (ошибка валидации бизнес-правил)
- **Окружение**: `https://qa-internship.avito.com`, дата: 2026-04-18
- **Шаги воспроизведения**:

- **Ожидаемый результат**:
  - `400 Bad Request`
